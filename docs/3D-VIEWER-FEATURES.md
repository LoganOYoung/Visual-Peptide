# 3D 查看器功能清单（梳理后）

## 保留（高价值 / 高使用率）

| 功能 | 说明 | 价值 |
|------|------|------|
| 残基点击/悬停 | 显示残基名、链、序号 | 核心，科研必备 |
| 热点解释 (Hotspots) | 悬停/点击显示药理/结构解释文案 | 差异化，专业感 |
| 链显示/隐藏 | 多链勾选显示 | 多链结构必备 |
| 显示模式 | Cartoon / Stick / Line / Sphere | 满足不同观察习惯 |
| 着色 | Spectrum / By chain | 常用 |
| 距离测量 | 两点选原子显示 Å + 连线 | 科研常用 |
| 序列与 3D 联动 | 点序列→3D 居中；点 3D→序列高亮 | 核对修饰位点 |
| 疏水表面 | 红蓝表面开关 | 理解溶解性，中高价值 |
| 导出 PNG | 一键下载当前视图 | 报告/PPT 常用 |
| 预设视图 | Focus all / Focus 某链 | 快速复位视角 |
| 元数据 + Cite/Download | 标题、分辨率、方法、链接 | 写论文/引用 |
| 可复现 URL | ?pdb=&chain=&residues=&labels= | 分享与复现 |
| Verify 链接 | 顶栏链到 /verify | 批次验证入口 |

## 已移除（低价值 / 低使用率 / 影响 UI）

| 功能 | 移除原因 |
|------|----------|
| Simple/Detailed 模式切换 | 使用率低，增加认知负担；统一为单一面板 |
| PNG (透明背景) | 使用率低于普通 PNG，减少按钮数量 |
| 5s 旋转录屏 (WebM) | 使用率低，浏览器兼容性差，实现复杂 |
| 结合演示 按钮 | 仅相机旋转，无真实对接内容，易被视为噱头 |
| pKi (pred.) 占位 | 无数据时占位干扰视线，有数据再上 |
| 二级结构图例 (α-helix · β-sheet · loop) | 信息密度低，占侧栏空间 |
| 结构页独立 Verify 区块 | 与顶栏 Verify 重复，保留一处即可 |

## 保留但可后续精简

- **Line** 显示：使用率低于 Cartoon/Stick，可考虑仅保留 Cartoon / Stick / Sphere。
- **fixedLabels / URL labels**：保留能力供分享链接使用，不在主 UI 暴露。
